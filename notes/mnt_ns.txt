Getting from `struct file` to mount namespace:

        file -> f_path -> vfsmnt -> mount -> mnt_namespace
                        We can use `container_of` for this

        file -> f_path -> vfsmnt
        container_of(vfsmnt, struct mount, mnt) -> mount
        mount -> mnt_namespace
